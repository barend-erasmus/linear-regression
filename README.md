# Linear Regression

In linear regression, the relationships are modeled using linear predictor functions whose unknown model parameters are estimated from the data. 


## Example

|     | Age (X) | Glucose Level (Y) | XY    | X^2   | Y^2   |

|     | 21      | 65                | 1365  | 441   | 4225  |
